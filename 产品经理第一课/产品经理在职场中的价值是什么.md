# 产品经理在职场中的价值是什么

产品经理在职场中的价值是

- 经营管理
  - 产品生命周期
  - 统筹产品的需求和功能定义，还有迭代优化
- 翻译（枢纽）
  - 战略（领导的发散性想法翻译成需求）
  - 立项（产品方案翻译成具体的产品功能）
  - 探讨（把产品功能翻译成产品细节）
  - 执行（把细节翻译成具体可执行的文档）
- 务实
  - 战略和战术（战略是指方向和提想法，战术是把提出来的想法开始具体的实施）
  - 管理和执行（老板在提想法的时候可能会忽略掉执行细节，在有新的想法的时候产品经理应该要好好规划产品开发的节奏，不要让团队工作过于混乱）
